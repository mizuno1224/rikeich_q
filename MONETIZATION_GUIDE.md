# Physics Lab. 収益化ガイド

このドキュメントでは、本サイト「Physics Lab.」を収益化するための手順と、実施済みのサイト整備内容をまとめています。

---

## 1. 収益化の選択肢（優先度順）

### A. Google AdSense（広告）【推奨・まずここから】
- **メリット**: 審査通過後はほぼ手間なしで継続収益。教育系・受験系は単価が比較的良いことが多い。
- **必要**: プライバシーポリシー、About/サイト紹介、十分なコンテンツ量、審査申請。
- **実施済み整備**: `privacy.html`、`about.html`、フッターリンク、広告枠用のHTML（`ad-slot`）。

**手順（簡潔）**  
1. [Google AdSense](https://www.google.com/adsense/) に登録し、サイトURLを追加。  
2. 審査用に「サイトの目的」「コンテンツの説明」を記入（About の内容を流用可）。  
3. 審査通過後、AdSense の「広告」→「広告ユニット」でコードを取得。  
4. 取得したコードを、`index.html` および `viewer.html` の `<!-- AD_SLOT_* -->` のコメント部分に貼り付け（または `js/ads.js` を作成して一元管理）。  
5. 再デプロイ後、数日〜数週間で表示・収益が反映されます。

---

### B. 投げ銭・寄付（Ko-fi / Buy Me a Coffee 等）
- **メリット**: 審査不要で即開始可能。ファンからの支援を得られる。
- **実施済み整備**: フッターに「支援する」リンク用の枠を用意。任意でURLを設定可能。

**手順**  
1. [Ko-fi](https://ko-fi.com/) や [Buy Me a Coffee](https://www.buymeacoffee.com/) でアカウント作成。  
2. 専用ページのURLを取得。  
3. フッターの「支援する」リンクの `href` をそのURLに変更（例: `footer.html` や共通フッターのリンク先）。

---

### C. 有料コンテンツ・会員制（将来オプション）
- **例**: 一部の「難関校解説」や「音声解説」を有料会員のみに公開。  
- **技術**: Firebase Authentication + Firestore で会員判定、または Stripe + 会員フラグで制御。  
- まずは **AdSense + 投げ銭** で運用し、コンテンツが増えてから検討するのがおすすめです。

## 2. 実施済みのサイト整備一覧

| 項目 | 内容 |
|------|------|
| **プライバシーポリシー** | `privacy.html` を追加。Cookie・アクセス解析・広告について記載。AdSense 申請前に必要。 |
| **About ページ** | `about.html` を追加。サイトの目的・対象者・使い方。審査・信頼性向上用。 |
| **フッター** | `index.html` と `viewer.html` に共通フッターを追加。「このサイトについて」「プライバシーポリシー」「支援する」へのリンク。 |
| **広告枠** | 広告用のプレースホルダー要素（`.ad-slot`）を設置。AdSense コードを貼るだけで表示可能。 |
| **メタ情報** | `index.html` の `description` を整備。SEO・審査用。 |

---

## 3. AdSense 審査を通過しやすくするために

- **コンテンツ量**: 解説ページをある程度の数（目安: 20〜30本以上）用意しておく。  
- **オリジナル**: 他サイトのコピーではなく、自作の解説・シミュレーションであることを明示（About に記載済み）。  
- **ナビ・リンク**: トップから主要コンテンツへ2クリック以内で到達できるようにする（現状のタブ・一覧で満たしやすい）。  
- **プライバシーポリシー**: 広告・Cookie に言及したページをフッターから必ず見られるようにする（実施済み）。  

---

## 4. 広告コードの貼り付け例（AdSense 審査通過後）

AdSense から「レスポンシブ」または「記事内」用のコードが発行されたら、次のように貼り付けます。

**例: index.html の広告枠**
```html
<!-- 広告枠: トップ下部（AdSense コードに差し替え） -->
<div class="ad-slot ad-slot--top">
  <!-- ここに AdSense の <ins class="adsbygoogle" ...> と script を貼る -->
</div>
```

**例: viewer.html の広告枠**
```html
<!-- 広告枠: 解説閲覧時（AdSense コードに差し替え） -->
<aside class="ad-slot ad-slot--sidebar" aria-label="広告">
  <!-- ここに AdSense のコードを貼る -->
</aside>
```

複数ページに同じコードを入れる場合は、`js/ads.js` を作成し、`data-ad-slot` や `data-ad-format` を変数で渡して 1 箇所で管理すると保守しやすくなります。

---

## 5. 注意事項

- **教材の著作権**: 問題文の転載は各出版社の許諾範囲内で行い、解説・図・シミュレーションはオリジナルにすること。  
- **個人情報**: フォームや会員機能を追加する場合は、プライバシーポリシーと利用規約を更新すること。  
- **広告の配置**: 広告が多すぎると審査で指摘されたり離脱が増えるため、まずは 1 ページ 1〜2 ユニット程度から始めることを推奨します。

---
