# Cursorでの解説ページ自動生成 - クイックスタート

## 5分で始める

### ステップ1: 画像・PDFをアップロード

Cursorのチャットに問題の画像やPDFをドラッグ&ドロップします。

### ステップ2: 教材情報を指定

以下のテンプレートをコピーして、情報を入力してください：

```
以下の問題について解説ページを作成してください。

【教材情報】
- 教材タイプ: [教材タイプを選択]
- [年度または章番号]: [数値]
- 問題番号: [問題番号]
- [単元名または大学名]: [該当する場合は入力]
- ファイル名ID: [半角英数]
```

### ステップ3: 生成と確認

Cursorが自動的に解説HTMLと登録JSONを生成します。生成されたコードを確認し、必要に応じて調整してください。

### ステップ4: ファイル保存

生成されたHTMLとJSONを適切な場所に保存します。マニフェストは自動更新されます。

## 使用例

### 例1: 共通テスト問題

```
以下の問題について解説ページを作成してください。

【教材情報】
- 教材タイプ: 共通テスト (exam_common)
- 年度: 2026
- 問題番号: 第1問
- ファイル名ID: 2026_q1
```

### 例2: 教科書問題

```
以下の問題について解説ページを作成してください。

【教材情報】
- 教材タイプ: 物理基礎 (textbook_basic)
- 編ID: 01
- 章ID: 01
- 問題番号: p.24 例1
- 単元名: 運動の表し方
- ファイル名ID: p24_ex1
```
