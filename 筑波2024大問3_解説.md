# 筑波大学 2024年度 物理 大問3 解説（薄膜干渉）

## 1. 実験の設定

- **図1・図2**: ガラス板Aを水平に置き、その左端にガラス板Bを接して傾ける。
- **座標**: ガラス板Bの左下の頂点を原点O(0,0)、右向きにx軸、鉛直下向きにy軸。
- **くさび形空気層**: x軸方向に距離Lの位置に厚さDの薄いフィルムを挟み、O付近で厚さ0、x=Lで厚さDのくさび形のすきま（空気層）ができる。
- **光**: 真上から単色光を鉛直下向きに入射し、真上に反射した光をスクリーンで観測。ガラス板の屈折率は \(n_G (>1)\)、空気の屈折率は 1。

---

## 問1：光路差 \(p\) を \(x,\,D,\,L\) で表せ

**考え方**  
位置 \(x\) での空気層の厚さは、くさびの相似から  
\[
d(x) = \frac{D}{L}\,x
\]  
である。  
ガラス板Bの下面で反射する光と、ガラス板Aの上面で反射する光の経路差は、空気層を往復する分で \(2d\)。空気中なので光路差はそのまま \(2d\)。

**答**
\[
\boxed{\ p = \frac{2D}{L}\,x\ }
\]

---

## 問2：\(m\) 番目の明線の位置 \(x_m\) を \(D,\,L,\,\lambda,\,m\) で表せ

**考え方**  
ガラス板Bの下面（空気→ガラス）とガラス板Aの上面（空気→ガラス）の反射のうち、実質的に一方だけが位相 \(\pi\) ずれるとみなすと、光路差に \(\lambda/2\) のずれが生じる。したがって**明線**の条件は \(2d + \dfrac{\lambda}{2} = m\lambda\)、すなわち
\[
2d = \left(m - \frac{1}{2}\right)\lambda \quad (m=1,2,3,\cdots)
\]  
\(d = \dfrac{D}{L}x\) より  
\[
2\,\frac{D}{L}\,x_m = \left(m - \frac{1}{2}\right)\lambda \quad \Rightarrow \quad x_m = \frac{\left(m - \frac{1}{2}\right)\lambda L}{2D}
\]

**答**
\[
\boxed{\ x_m = \frac{\left(m - \frac{1}{2}\right)L\lambda}{2D}\ }
\]

---

## 問3：明線の間隔 \(\Delta x\) を \(D,\,L,\,\lambda\) で表せ

**考え方**  
隣り合う明線は \(m\) が 1 ずれるので（問2の \(x_m\) を用いて）  
\[
\Delta x = x_{m+1} - x_m = \frac{\left((m+1) - \frac{1}{2}\right)L\lambda}{2D} - \frac{\left(m - \frac{1}{2}\right)L\lambda}{2D} = \frac{L\lambda}{2D}
\]

**答**
\[
\boxed{\ \Delta x = \frac{L\lambda}{2D}\ }
\]

---

## 問4：ガラス板Aを \(y\) 軸正方向に平行移動した後の \(m\) 番目の明線の位置 \(x'_m\)

**設定**  
ガラス板Bとフィルムは固定。ガラス板Aだけを \(y\) 軸の正の方向（下向き）に平行移動する。  
ガラス板Aの左上の頂点を点Pとし、移動後のPの座標を \((0,\,y)\) とする。

**考え方**  
移動後、位置 \(x\) での**空気層の厚さ**は \(d'(x) = \dfrac{D}{L}\,x - y\) である。明線の条件は問2と同様 \(2d' = \left(m - \dfrac{1}{2}\right)\lambda\) なので、  
\[
2\left(\frac{D}{L}\,x'_m - y\right) = \left(m - \frac{1}{2}\right)\lambda
\]  
これを解いて  
\[
x'_m = \frac{L}{2D}\left\{\left(m - \frac{1}{2}\right)\lambda - 2y\right\}
\]  
\(y\) が増える（Aを下に動かす）と \(\left(m - \dfrac{1}{2}\right)\lambda - 2y\) は小さくなるので、\(x'_m\) は小さくなり、**明線は \(x\) 軸の負の方向に移動する**。

**答**
\[
\boxed{\ x'_m = \frac{L}{2D}\left\{\left(m - \frac{1}{2}\right)\lambda - 2y\right\}\ }
\]

（問題の条件 \(0 < x'_m < L\) を満たす \(m\) を考えている。）

## 問5：明線が移動した方向（\(x\) 軸の正・負）

**考え方**  
Aを下に動かすと、同じ \(x\) では空気層の厚さ \(d' = \dfrac{D}{L}x - y\) は \(y\) の増加で小さくなる。  
同じ \(m\) 番目の明線（同じ \(d' = \dfrac{\left(m - \frac{1}{2}\right)\lambda}{2}\)）を見るには、\(d'\) を一定に保つために \(x\) を**小さく**する必要がある。  
つまり、**明線は \(x\) 軸の負の方向に移動する**。

**答**
\[
\boxed{\ \text{負の方向}\ }
\]

---

## 問6：点Pの座標 \((0,\,Y)\) のとき、\(m\) 番目の明線が水平方向にちょうど4本ずれたときの \(Y\)

**考え方**  
問5より、Aを下に動かすと明線は \(x\) 軸の負の方向にずれる。したがって「4本ずれた」は、**移動後の \(m\) 番目の明線の位置**が、**移動前の \((m-4)\) 番目の明線の位置**と一致した、と解釈する。

移動前：\(x_{m-4} = \dfrac{\left(m - 4 - \frac{1}{2}\right)L\lambda}{2D} = \dfrac{\left(m - \frac{9}{2}\right)L\lambda}{2D}\)  
移動後：\(x'_m = \dfrac{L}{2D}\left\{\left(m - \frac{1}{2}\right)\lambda - 2Y\right\}\)

これらが等しいので  
\[
\frac{L}{2D}\left\{\left(m - \frac{1}{2}\right)\lambda - 2Y\right\} = \frac{\left(m - \frac{9}{2}\right)L\lambda}{2D}
\]  
\[
\left(m - \frac{1}{2}\right)\lambda - 2Y = \left(m - \frac{9}{2}\right)\lambda \quad \Rightarrow \quad -2Y = -4\lambda \quad \Rightarrow \quad Y = 2\lambda
\]

**答**
\[
\boxed{\ Y = 2\lambda\ }
\]

---

## 問7：液体で満たしたときの明線の間隔 \(\Delta X\)（波長 \(\lambda'\)、液体の屈折率 \(n\)）

**設定**  
ガラス板A・Bとフィルムで囲まれたくさび形の空間を屈折率 \(n\) の液体で満たし、波長 \(\lambda'\)（\(>\lambda\)）の単色光を入射。液体の屈折率 \(n\) は波長に依存しないとする。

**考え方**  
空気層の代わりに液体が入るので、**光路差**は厚さ \(d\) に対して \(2nd\)。真空中の波長 \(\lambda'\) で、明線の条件は問2と同様の扱いで \(2nd = \left(m - \dfrac{1}{2}\right)\lambda'\) となり、間隔は  
\[
\Delta X = x_{m+1} - x_m = \frac{L\lambda'}{2nD}
\]

**答**
\[
\boxed{\ \Delta X = \frac{L\lambda'}{2nD}\ }
\]

---

## 問8：\(\Delta X = \Delta x\) のときの液体の屈折率 \(n\)

**考え方**  
問3より \(\Delta x = \dfrac{L\lambda}{2D}\)、問7より \(\Delta X = \dfrac{L\lambda'}{2nD}\)。  
\(\Delta X = \Delta x\) より  
\[
\frac{L\lambda'}{2nD} = \frac{L\lambda}{2D} \quad \Rightarrow \quad \frac{\lambda'}{n} = \lambda \quad \Rightarrow \quad n = \frac{\lambda'}{\lambda}
\]

**答**
\[
\boxed{\ n = \frac{\lambda'}{\lambda}\ }
\]

---

## まとめ

| 問 | 答 |
|----|-----|
| 問1 | \(p = \dfrac{2Dx}{L}\) |
| 問2 | \(x_m = \dfrac{\left(m - \frac{1}{2}\right)L\lambda}{2D}\) |
| 問3 | \(\Delta x = \dfrac{L\lambda}{2D}\) |
| 問4 | \(x'_m = \dfrac{L}{2D}\left\{\left(m - \dfrac{1}{2}\right)\lambda - 2y\right\}\) |
| 問5 | 負の方向 |
| 問6 | \(Y = 2\lambda\) |
| 問7 | \(\Delta X = \dfrac{L\lambda'}{2nD}\) |
| 問8 | \(n = \dfrac{\lambda'}{\lambda}\) |

**ポイント**
- くさび形の厚さは \(d = \dfrac{D}{L}x\) で線形。
- 反射の位相のずれを考慮すると明線は \(2d = \left(m - \dfrac{1}{2}\right)\lambda\)。
- Aを下に動かすと厚さが減るので、同じ \(m\) の明線は左（\(x\) 軸の負の方向）へ移動。
- 液体を入れると光路差は \(2nd\) から間隔 \(\Delta X = \dfrac{L\lambda'}{2nD}\)。\(\Delta X = \Delta x\) のとき \(n = \dfrac{\lambda'}{\lambda}\)。
