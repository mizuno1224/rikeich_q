大問1：台上の振り子の運動（運動量保存・エネルギー保存）

解法の指針

この問題は，滑らかな床の上に置かれた台と，その台に取り付けられた振り子という系を扱います。台が固定されている場合と自由に動ける場合で，物理現象が大きく異なります。

問題の核心

  • 問(1)：台が固定されている場合 → 単純なエネルギー保存と円運動の問題
  • 問(2)：台上の観測者から見た振り子運動 → 慣性系と非慣性系の切り替え
  • 問(3)：台が自由に動く場合 → 水平方向の運動量保存と力学的エネルギー保存の連立
  • 問(4)〜(6)：撃力を与えた後の運動 → 運動量保存とエネルギー保存の適用
キーポイント
外力（床からの摩擦力）が水平方向に働かないため，系全体の水平方向の運動量は保存されます。これが本問を解く最大の鍵です。

(1) 台が固定された場合の単振り子

問題設定

台が動かない（固定されている）場合を考えます。糸が鉛直方向から左に角度 $\theta$ 傾いたところで小球を静止させてから静かに放します。

小球の速度の導出

台が固定されているため，これは単純な単振り子の問題です。 力学的エネルギー保存則を適用します。

最下点を基準とした高さは $h = l(1 - \cos\theta)$ です。
初期状態（静止）と最下点での力学的エネルギー保存：

$$mgl(1 - \cos\theta) + 0 = 0 + \frac{1}{2}mv^2$$

整理すると：

$$v = \sqrt{2gl(1-\cos\theta)}$$

糸の張力の導出

最下点では，小球は円運動をしています。向心方向（鉛直上向き）の運動方程式を立てます：

$$T - mg = \frac{mv^2}{l}$$

$v^2 = 2gl(1-\cos\theta)$ を代入すると：

$$T = mg + \frac{m \cdot 2gl(1-\cos\theta)}{l} = mg + 2mg(1-\cos\theta)$$ $$T = mg(3 - 2\cos\theta)$$
答え：
小球の速度：$$v = \sqrt{2gl(1 - \cos\theta)}$$ 糸の張力：$$T = mg(3 - 2\cos\theta)$$

別解：半角公式を用いた表現

三角関数の公式 $1 - \cos\theta = 2\sin^2\dfrac{\theta}{2}$ を用いると：

$$v = \sqrt{2gl \cdot 2\sin^2\frac{\theta}{2}} = 2\sqrt{gl}\sin\frac{\theta}{2}$$

この形は，$\theta$ が小さいとき $v \approx \sqrt{gl}\cdot\theta$ となり，微小振動の結果と整合します。

(2) 台上の観測者から見た振り子運動

問題設定

小球を最下点に静止させた状態から，ゆっくり台を右向きに加速度 $a$ で加速します。その後瞬時に加速をやめて等速運動させると，台上で小球は振り子運動をします。台に静止した観測者から見たとき，運動中の小球の速度の最大値を求めます。

解法のポイント

「ゆっくり加速」とは，小球が準静的に（振動せずに）釣り合いを保ちながら傾いていくことを意味します。このとき，台に対して小球は静止しています。

ステップ1：加速中の小球の釣り合い角度

台と一緒に加速度 $a$ で動く観測者（非慣性系）から見ると，小球には見かけの力（慣性力）$ma$ が左向きに働きます。

小球が角度 $\phi$ で釣り合うとき（水平方向と鉛直方向）：

$$T\sin\phi = ma \quad \cdots (1)$$ $$T\cos\phi = mg \quad \cdots (2)$$

(1)÷(2)より：

$$\tan\phi = \frac{a}{g}$$

ステップ2：等速運動に切り替わった瞬間

加速をやめて等速運動に切り替わると，慣性力が消えます。小球は角度 $\phi$ の位置から振り子運動を始めます。

台に対して静止した観測者から見ると，台は慣性系になるので，通常の単振り子と同じです。最下点で速度が最大になります。

ステップ3：エネルギー保存

高さの差は $h = l(1 - \cos\phi)$ です。エネルギー保存より：

$$mgl(1 - \cos\phi) = \frac{1}{2}mv_{\max}^2$$ $$v_{\max} = \sqrt{2gl(1 - \cos\phi)}$$

ステップ4：$\cos\phi$ を $a, g$ で表す

$\tan\phi = a/g$ より，$\cos\phi = \frac{g}{\sqrt{a^2 + g^2}}$ です。

$$1 - \cos\phi = 1 - \frac{g}{\sqrt{a^2 + g^2}} = \frac{\sqrt{a^2 + g^2} - g}{\sqrt{a^2 + g^2}}$$

したがって：

$$v_{\max} = \sqrt{2gl \cdot \frac{\sqrt{a^2 + g^2} - g}{\sqrt{a^2 + g^2}}}$$
答え：
$$v_{\max} = \sqrt{2gl\left(1 - \frac{g}{\sqrt{a^2 + g^2}}\right)}$$

別解：有理化による簡略化

分子分母に $\sqrt{a^2+g^2}+g$ を掛けて有理化すると：

$$1 - \frac{g}{\sqrt{a^2+g^2}} = \frac{a^2}{\sqrt{a^2+g^2}(\sqrt{a^2+g^2}+g)}$$

よって：

$$v_{\max} = a\sqrt{\frac{2l}{\sqrt{a^2+g^2}+g}}$$

別解：見かけの重力を使う方法

加速中の台上では，見かけの重力 $g' = \sqrt{a^2 + g^2}$ が $\phi$ 方向に働くと考えることができます。等速運動に切り替わった後は $g$ に戻りますが，$\phi$ の位置から放したことと同じなので：

$$v_{\max} = \sqrt{2gl(1 - \cos\phi)}$$

ここで $\cos\phi = g/g' = g/\sqrt{a^2+g^2}$ を代入すれば同じ結果が得られます。

(3) 台が自由に動く場合（θ = 60°）

問題設定

静止した台の上で，糸が鉛直方向から左に角度 $\theta = 60°$ 傾いたところで小球を静止させてから静かに放します。床に静止した観測者から見たとき，小球が最下点に達したときの小球の速度と台の速度，および糸の張力を求めます。

解法の核心：2つの保存則

床は滑らかなので，系に対して水平方向の外力は働きません。よって：

  1. 水平方向の運動量保存：$mv_{\text{球,x}} + MV = 0$（初期運動量は0）
  2. 力学的エネルギー保存：位置エネルギーの減少 = 運動エネルギーの増加

ステップ1：最下点での速度の関係

最下点では糸は鉛直なので，小球は台に対して水平方向にのみ運動します。床から見た小球の水平速度を $v$（右向き正），台の速度を $V$ とします。

運動量保存則より：

$$mv + MV = 0$$ $$V = -\frac{m}{M}v \quad \cdots (1)$$

（台は小球と逆向きに動く）

ステップ2：エネルギー保存

小球の高さの減少は $h = l(1 - \cos 60°) = l(1 - 1/2) = l/2$ です。

$$mg \cdot \frac{l}{2} = \frac{1}{2}mv^2 + \frac{1}{2}MV^2$$

(1)を代入：

$$\frac{mgl}{2} = \frac{1}{2}mv^2 + \frac{1}{2}M\left(\frac{m}{M}v\right)^2 = \frac{1}{2}mv^2 + \frac{m^2v^2}{2M}$$ $$\frac{mgl}{2} = \frac{1}{2}mv^2\left(1 + \frac{m}{M}\right) = \frac{1}{2}mv^2 \cdot \frac{M+m}{M}$$

整理すると：

$$v^2 = \frac{Mgl}{M+m}$$ $$v = \sqrt{\frac{Mgl}{M+m}}$$

ステップ3：台の速度

(1)より：

$$V = -\frac{m}{M}\sqrt{\frac{Mgl}{M+m}} = -\sqrt{\frac{m^2gl}{M(M+m)}}$$

大きさで表すと：

$$|V| = m\sqrt{\frac{gl}{M(M+m)}}$$

解法2：重心に注目した解法（推奨）

系全体の重心は水平方向に動きません。この性質を使うと見通しよく解けます。

初期状態で重心の位置を原点とします。台の位置を $X$，小球の位置を $x$ とすると：

$$MX + mx = 0 \quad \text{（重心が原点に固定）}$$

初期状態：台は原点付近，小球は左に $l\sin 60° = \dfrac{\sqrt{3}}{2}l$ だけずれています。

最下点：小球は台の真下にいるので，$x = X$（小球と台の水平位置が一致）。重心条件より：

$$MX + mX = 0 \quad \Rightarrow \quad X = 0$$

つまり最下点では台も小球も重心位置（原点）にいます。

重心条件 $MX + mx = 0$ を時間微分すると：

$$M\dot{X} + m\dot{x} = 0 \quad \Rightarrow \quad MV + mv = 0$$

これは運動量保存則そのものです！

ステップ4：糸の張力

最下点では，小球は台に対して円運動をしています。台から見た小球の相対速度を求めます。

$$v_{\text{rel}} = v - V = v + \frac{m}{M}v = v \cdot \frac{M+m}{M} = \sqrt{\frac{Mgl}{M+m}} \cdot \frac{M+m}{M} = \sqrt{\frac{(M+m)gl}{M}}$$

最下点で台に働く水平方向の力は0（糸は鉛直）なので，台の加速度は0です。よって床から見た向心加速度は：

$$a_c = \frac{v_{\text{rel}}^2}{l} = \frac{(M+m)g}{M}$$

小球の鉛直方向の運動方程式：

$$T - mg = ma_c = m \cdot \frac{(M+m)g}{M} = \frac{m(M+m)g}{M}$$ $$T = mg + \frac{m(M+m)g}{M} = mg\left(1 + \frac{M+m}{M}\right) = mg \cdot \frac{2M+m}{M}$$ $$T = \frac{(2M+m)mg}{M}$$
答え：
小球の速度（右向き）：$$v = \sqrt{\frac{Mgl}{M+m}}$$ 台の速度（左向き）：$$V = -m\sqrt{\frac{gl}{M(M+m)}} \quad \text{（大きさ：} m\sqrt{\frac{gl}{M(M+m)}}\text{）}$$ 糸の張力：$$T = \frac{(2M+m)mg}{M}$$

補足：重心の運動

系全体の重心は水平方向に静止しています（初期運動量が0のため）。シミュレーションで緑の点線が動かないことを確認してください。これは運動量保存の視覚的表現です。

(4) 撃力を与えた場合：糸が水平になったときの台の速度

問題設定

静止した台の上で小球を最下点で静止させた後，撃力により台に水平右方向の初速度 $V_0$ を瞬時に与えます。小球が糸が水平になる高さまで達したとき，台の速度を求めます。

撃力直後の初期条件

  • 台の速度：$V_0$（右向き）
  • 小球の速度：$0$（撃力は台のみに作用）
  • 初期運動量：$p_0 = MV_0$

解法1：拘束条件を使う方法

糸が水平になったとき，台の速度を $V$，小球の速度を $(v_x, v_y)$ とします。

運動量保存：

$$MV + mv_x = MV_0 \quad \cdots (1)$$

拘束条件：糸が伸びないので，糸方向の相対速度成分は0です。

糸が水平のとき，糸方向は水平方向なので：

$$v_x - V = 0 \quad \Rightarrow \quad v_x = V \quad \cdots (2)$$

(2)を(1)に代入：

$$(M + m)V = MV_0$$ $$V = \frac{MV_0}{M+m}$$

解法2：重心に注目した解法

撃力直後の運動量は $MV_0$ なので，重心の速度は：

$$v_G = \frac{MV_0}{M+m}$$

糸が水平のとき，小球と台の水平速度が等しい（$v_x = V$）ということは，両者が重心と同じ速度で動いていることを意味します。

したがって，直ちに：

$$V = v_G = \frac{MV_0}{M+m}$$
答え：
$$V = \frac{MV_0}{M+m}$$

(5) 糸が水平になったときの小球の速度の大きさ

エネルギー保存則の適用

初期状態（撃力直後）と糸が水平になった状態でエネルギー保存を適用します。

初期状態（最下点，撃力直後）

  • 台の速度：$V_0$
  • 小球の速度：$0$
  • 小球の高さ：$0$（基準）
$$E_{\text{初}} = \frac{1}{2}MV_0^2$$

糸が水平になったとき

  • 台の速度：$V = \frac{MV_0}{M+m}$（(4)の結果）
  • 小球の水平速度：$v_x = V = \frac{MV_0}{M+m}$
  • 小球の鉛直速度：$v_y$（未知）
  • 小球の高さ：$l$
$$E_{\text{終}} = \frac{1}{2}MV^2 + \frac{1}{2}m(v_x^2 + v_y^2) + mgl$$

$v_x = V$ なので：

$$E_{\text{終}} = \frac{1}{2}(M+m)V^2 + \frac{1}{2}mv_y^2 + mgl$$

エネルギー保存

$$\frac{1}{2}MV_0^2 = \frac{1}{2}(M+m)V^2 + \frac{1}{2}mv_y^2 + mgl$$ $$\frac{1}{2}MV_0^2 = \frac{1}{2}(M+m)\left(\frac{MV_0}{M+m}\right)^2 + \frac{1}{2}mv_y^2 + mgl$$ $$\frac{1}{2}MV_0^2 = \frac{M^2V_0^2}{2(M+m)} + \frac{1}{2}mv_y^2 + mgl$$

整理すると：

$$\frac{1}{2}mv_y^2 = \frac{1}{2}MV_0^2 - \frac{M^2V_0^2}{2(M+m)} - mgl = \frac{MV_0^2(M+m) - M^2V_0^2}{2(M+m)} - mgl$$ $$= \frac{MmV_0^2}{2(M+m)} - mgl$$ $$v_y^2 = \frac{MV_0^2}{M+m} - 2gl$$

小球の速度の大きさ

$$|\vec{v}|^2 = v_x^2 + v_y^2 = \left(\frac{MV_0}{M+m}\right)^2 + \frac{MV_0^2}{M+m} - 2gl$$ $$= \frac{M^2V_0^2 + MV_0^2(M+m)}{(M+m)^2} - 2gl = \frac{MV_0^2(2M+m)}{(M+m)^2} - 2gl$$

したがって：

$$|\vec{v}| = \sqrt{\frac{M(2M+m)V_0^2}{(M+m)^2} - 2gl}$$
答え：
$$|\vec{v}| = \sqrt{\frac{M(2M+m)V_0^2}{(M+m)^2} - 2gl}$$

別解：重心系での計算（計算が楽）

重心系で考えると計算が簡単になります。重心速度は $V_{\text{com}} = \frac{MV_0}{M+m}$ です。

重心系での初期状態：

  • 台の速度：$V_0 - V_{\text{com}} = V_0 - \frac{MV_0}{M+m} = \frac{mV_0}{M+m}$（右向き）
  • 小球の速度：$0 - V_{\text{com}} = -\frac{MV_0}{M+m}$（左向き）

重心系での初期運動エネルギー：

$$E_{\text{com,初}} = \frac{1}{2}M\left(\frac{mV_0}{M+m}\right)^2 + \frac{1}{2}m\left(\frac{MV_0}{M+m}\right)^2 = \frac{MmV_0^2}{2(M+m)}$$

重心系での糸が水平になったとき：

  • 台の速度：$V - V_{\text{com}} = 0$（重心系では台は静止）
  • 小球の水平速度：$v_x - V_{\text{com}} = 0$（重心系では小球の水平速度も0）
  • 小球の鉛直速度：$v_y$（重心系でも同じ）

重心系での運動エネルギー：

$$E_{\text{com,終}} = \frac{1}{2}mv_y^2$$

重心系では位置エネルギーは変わらないので（重心は等速運動），エネルギー保存より：

$$\frac{MmV_0^2}{2(M+m)} = \frac{1}{2}mv_y^2$$ $$v_y^2 = \frac{MV_0^2}{M+m}$$

床から見た小球の速度：

$$|\vec{v}|^2 = v_x^2 + v_y^2 = \left(\frac{MV_0}{M+m}\right)^2 + \frac{MV_0^2}{M+m} = \frac{MV_0^2}{(M+m)^2}(M + M + m) - 2gl$$ $$= \frac{M(2M+m)V_0^2}{(M+m)^2} - 2gl$$

ただし，位置エネルギーの変化 $mgl$ を考慮する必要があります。重心系でのエネルギー保存を正確に適用すると：

$$\frac{MmV_0^2}{2(M+m)} = \frac{1}{2}mv_y^2 + mgl$$ $$v_y^2 = \frac{MV_0^2}{M+m} - 2gl$$

これより同じ結果が得られます。重心系を使うと，水平方向の運動が自動的に分離され，計算が簡潔になります。

(6) 初速度 $V_0$ の最小値

糸が水平に達する条件

小球が糸が水平になる高さに達するためには，その高さで鉛直速度成分が実数でなければなりません。(5)の結果より：

$$v_y^2 = \frac{MV_0^2}{M+m} - 2gl \geq 0$$

最小条件

ちょうど糸が水平になる高さに達する（その瞬間に $v_y = 0$）のは：

$$\frac{MV_0^2}{M+m} = 2gl$$ $$V_0^2 = \frac{2gl(M+m)}{M}$$ $$V_0 = \sqrt{\frac{2gl(M+m)}{M}}$$
答え：
$$V_{0,\min} = \sqrt{\frac{2gl(M+m)}{M}} = \sqrt{2gl\left(1 + \frac{m}{M}\right)}$$

物理的解釈

この結果は直感的にも理解できます：

  • $M \gg m$（台が非常に重い）のとき：$V_{0,\min} \approx \sqrt{2gl}$（台が動かない場合と同じ）
  • $M = m$（台と小球が同じ質量）のとき：$V_{0,\min} = 2\sqrt{gl}$（台が動くため余分なエネルギーが必要）
  • $M < m$（台が軽い）のとき：さらに大きな初速度が必要

別解1：エネルギーの観点から直接導出

糸がちょうど水平に達する条件は，その瞬間に小球の鉛直速度が0になることです。このとき小球と台の水平速度は等しく $V$ です。

エネルギー保存：

$$\frac{1}{2}MV_0^2 = \frac{1}{2}(M+m)V^2 + mgl$$

運動量保存（$v_x = V$ を使用）：

$$MV_0 = (M+m)V \quad \Rightarrow \quad V = \frac{MV_0}{M+m}$$

代入：

$$\frac{1}{2}MV_0^2 = \frac{1}{2}(M+m)\left(\frac{MV_0}{M+m}\right)^2 + mgl$$ $$\frac{1}{2}MV_0^2 = \frac{M^2V_0^2}{2(M+m)} + mgl$$ $$\frac{MV_0^2(M+m) - M^2V_0^2}{2(M+m)} = mgl$$ $$\frac{MmV_0^2}{2(M+m)} = mgl$$ $$V_0^2 = \frac{2gl(M+m)}{M}$$

これは運動量保存とエネルギー保存から直接得られる結果です。

別解2：重心系での計算（計算が楽）

重心系で考えると計算が非常に簡単になります。重心速度は $V_{\text{com}} = \frac{MV_0}{M+m}$ です。

重心系での初期状態：

  • 台の速度：$\frac{mV_0}{M+m}$（右向き）
  • 小球の速度：$-\frac{MV_0}{M+m}$（左向き）

重心系での初期運動エネルギー：

$$E_{\text{com,初}} = \frac{1}{2}M\left(\frac{mV_0}{M+m}\right)^2 + \frac{1}{2}m\left(\frac{MV_0}{M+m}\right)^2 = \frac{MmV_0^2}{2(M+m)}$$

重心系での糸が水平になったとき：

  • 台の速度：$0$（重心系では台は静止）
  • 小球の水平速度：$0$（重心系では小球の水平速度も0）
  • 小球の鉛直速度：$0$（ちょうど水平に達する条件）

重心系での運動エネルギー：$0$

重心系では位置エネルギーは変わらないので（重心は等速運動），エネルギー保存より：

$$\frac{MmV_0^2}{2(M+m)} = mgl$$ $$V_0^2 = \frac{2gl(M+m)}{M}$$

重心系を使うと，水平方向の運動が自動的に分離され，計算が非常に簡潔になります。特に(5)と(6)の両方で重心系を使うと，一貫性があり理解しやすくなります。

Point

解法のポイントまとめ

  • 水平方向の運動量保存：床が滑らかなとき，系の水平方向の運動量は保存される。これが本問を解く最大の鍵です。
  • 重心の運動に注目：外力がなければ重心は静止（または等速運動）。台と小球の位置関係を見通しよく把握できる。
  • 拘束条件の活用：糸が伸びないことから，糸方向の相対速度成分は0。特に糸が水平のとき $v_x = V$。
  • 非慣性系での見かけの力：加速する台上では慣性力を考慮。見かけの重力の方向が傾く。
