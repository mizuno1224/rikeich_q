大問1：2球の回転運動と相対運動

解法の全体像

なめらかな水平面上を糸でつながれて運動する2つの小球の問題です。 重心運動と相対運動を分けて考えることがポイントです。

  • [I] 重心と相対速度：重心の位置と、重心から見た各小球の運動（相対運動）を考えます。相対運動は等速円運動になります。
  • [II] 図2の瞬間の解析：特定の初期条件における重心の運動と、重心まわりの円運動を組み合わせます。
  • [III] 図3の状態：重心運動と円運動の位相（回転角）を考慮して、小球の位置座標を求めます。

[I] 基本設定と保存則

(ア) 重心GからAまでの距離

小球A（質量 \(m_1\)）と小球B（質量 \(m_2\)）の重心Gの位置を考えます。 AとBの距離は \(L\) です。重心の定義より、GはABを \(m_2 : m_1\) に内分する点になります。 したがって、GからAまでの距離 \(r_1\) は、 \[ r_1 = \frac{m_2}{m_1+m_2} L \] となります。

答え：(ア)
\[ \frac{m_2}{m_1+m_2} L \]

(イ) 相対速度の関係

A, Bの速度はそれぞれ \(\vec{v_G} + \vec{v_1}\), \(\vec{v_G} + \vec{v_2}\) と表されます。 全運動量は \[ m_1(\vec{v_G} + \vec{v_1}) + m_2(\vec{v_G} + \vec{v_2}) = (m_1+m_2)\vec{v_G} + (m_1\vec{v_1} + m_2\vec{v_2}) \] と書けます。

Aが糸から受ける力とBが糸から受ける力は、作用・反作用の関係で大きさが同じ・向きが逆なので、系に働く外力の和は 0 であり、運動量保存則より系の全運動量は一定です。重心の定義から、全運動量は常に「全質量 \(\times\) 重心の速度」に等しくなります（\(\vec{P} = (m_1+m_2)\vec{v_G}\)）。よって全運動量 \((m_1+m_2)\vec{v_G} + (m_1\vec{v_1} + m_2\vec{v_2})\) が \((m_1+m_2)\vec{v_G}\) に等しく、 \[ m_1\vec{v_1} + m_2\vec{v_2} = \vec{0} \quad \Rightarrow \quad \vec{v_2} = -\frac{m_1}{m_2}\vec{v_1} \] が成り立ちます。よって \(\vec{v_2} = \textbf{(イ)}\, \vec{v_1}\) の係数は次のとおりです。

答え：(イ)
\[ -\frac{m_1}{m_2} \]

(a) 床から見た重心Gの運動

問われていること：空欄(a)は「床（慣性系）から見た重心Gの運動」が何かを答える問題です。Gそのものの動き方です。

系に働く外力はありません（糸の張力はAとBの間の内力）。したがって重心Gには合力が働かず、床から見てGは等速直線運動をします。（(1)の等速円運動は、Gから見たA・Bの相対運動のことであり、ここで聞かれている「Gの運動」とは別です。）

答え：(a)
(2) 等速直線運動

[II] 図2の瞬間の解析

図2の状態では、Aは原点Oにあり \(\vec{v_A} = \vec{0}\)、Bは \(x=L\) にあり \(\vec{v_B} = (0, v_0)\) です。

(ウ) 重心Gの速さ

重心Gの速度 \(\vec{v_G}\) は \[ \vec{v_G} = \frac{m_1 \vec{v_A} + m_2 \vec{v_B}}{m_1+m_2} = \frac{m_1 \vec{0} + m_2 (0, v_0)}{m_1+m_2} = \left( 0, \frac{m_2 v_0}{m_1+m_2} \right) \] よって、重心Gの速さは次のとおりです。

答え：(ウ)
\[ \frac{m_2 v_0}{m_1+m_2} \]

(エ) GからみたBの円運動の角速度

重心GからBまでの距離（回転半径）は \(r_2 = L - r_1 = L - \frac{m_2 L}{m_1+m_2} = \frac{m_1 L}{m_1+m_2}\) です。 BのGに対する相対速度 \(\vec{v_2}\) は \[ \vec{v_2} = \vec{v_B} - \vec{v_G} = (0, v_0) - \left( 0, \frac{m_2 v_0}{m_1+m_2} \right) = \left( 0, \frac{m_1 v_0}{m_1+m_2} \right) \] 相対速さ \(v_2 = |\vec{v_2}| = \frac{m_1 v_0}{m_1+m_2}\) です。 円運動の式 \(v_2 = r_2 \omega\) より、角速度 \(\omega\) は \[ \omega = \frac{v_2}{r_2} = \frac{\frac{m_1 v_0}{m_1+m_2}}{\frac{m_1 L}{m_1+m_2}} = \frac{v_0}{L} \]

答え：(エ)
\[ \frac{v_0}{L} \]

(オ) Aが糸から受ける力の大きさ

AはGを中心とする半径 \(r_1\) の等速円運動をしています。張力 \(T\) が向心力となるので、 \[ T = m_1 r_1 \omega^2 = m_1 \left( \frac{m_2 L}{m_1+m_2} \right) \left( \frac{v_0}{L} \right)^2 = \frac{m_1 m_2}{m_1+m_2} \frac{v_0^2}{L} \]

答え：(オ)
\[ \frac{m_1 m_2}{m_1+m_2} \frac{v_0^2}{L} \]

(カ) 相対運動の運動エネルギーの和

求める値は \(K_{\text{rel}} = \frac{1}{2}m_1|\vec{v_1}|^2 + \frac{1}{2}m_2|\vec{v_2}|^2\) です。 \(|\vec{v_1}| = r_1 \omega\), \(|\vec{v_2}| = r_2 \omega\) なので、 \[ K_{\text{rel}} = \frac{1}{2}\omega^2 (m_1 r_1^2 + m_2 r_2^2) \] 括弧内 \(m_1 r_1^2 + m_2 r_2^2\)（重心のまわりの回転のしやすさを表す量で、大学では慣性モーメントと呼ばれる）を計算すると、 \[ m_1 r_1^2 + m_2 r_2^2 = m_1 \left( \frac{m_2 L}{m_1+m_2} \right)^2 + m_2 \left( \frac{m_1 L}{m_1+m_2} \right)^2 = \frac{m_1 m_2 L^2}{(m_1+m_2)^2} (m_2 + m_1) = \frac{m_1 m_2 L^2}{m_1+m_2} \] したがって \[ K_{\text{rel}} = \frac{1}{2} \left( \frac{v_0}{L} \right)^2 \frac{m_1 m_2 L^2}{m_1+m_2} = \frac{1}{2} \frac{m_1 m_2}{m_1+m_2} v_0^2 \]

答え：(カ)
\[ \frac{1}{2} \frac{m_1 m_2}{m_1+m_2} v_0^2 \]

[III] 図3の状態（回転と移動）

重心Gは初速度のまま等速直線運動を続け、AとBはGのまわりを等速円運動します。 初期状態（図2）では、Aは原点Oに、Gは \((a, 0)\) に、Bは \((L, 0)\) にありました（ここで \(a = r_1\) とします）。

重心Gの速度は \(\vec{v_G} = (0, a\omega)\) です（\(\because V_G = \frac{m_2 v_0}{m_1+m_2} = r_1 \frac{v_0}{L}\frac{m_1+m_2}{m_2} \frac{m_2}{m_1+m_2} = a\omega\)）。 したがって時刻 \(t\) におけるGの位置は \((a, a\omega t)\) です。

回転角の考察

図2では \(\vec{GB}\) の偏角は \(0\)、\(\vec{GA}\) の偏角は \(\pi\) です。 これらが角速度 \(\omega\) で反時計回りに回転します。 「BがGからみて2回転し、3回転目の途中で図3のように糸とx軸のなす角が \(\frac{\pi}{4}\) となった」状況を考えます。

図3では、Aが右上、Bが左下に描かれており、線分AB（糸）の角度が \(\frac{\pi}{4}\) です。 つまり、ベクトル \(\vec{GA}\) の偏角は \(\frac{\pi}{4}\) （または \(2n\pi + \frac{\pi}{4}\)）です。 初期の \(\vec{GA}\) の偏角は \(\pi\) でした。 ここから2回転（\(4\pi\)）以上回転して \(\frac{\pi}{4}\) の向きになるまでの回転角 \(\Delta \theta\) は、 \[ \Delta \theta = (\text{2回転}) + (\pi \to \frac{\pi}{4} \text{の回転}) \] \(\pi\) から反時計回りに \(\frac{\pi}{4}\) の向きになるには、\(\pi \to 2\pi \to 2\pi+\frac{\pi}{4}\) なので、追加の回転は \(\pi + \frac{\pi}{4} = \frac{5\pi}{4}\) です。 よって総回転角は \[ \omega t = 4\pi + \frac{5\pi}{4} = \frac{21\pi}{4} \] となります。

Aの座標

このときのAの位置 \(\vec{r_A}\) は、\(\vec{r_G} + \vec{GA}\) です。 \[ \vec{r_G} = (a, a\omega t) = \left( a, a \cdot \frac{21\pi}{4} \right) \] \[ \vec{GA} = (a \cos \frac{\pi}{4}, a \sin \frac{\pi}{4}) = \left( \frac{a}{\sqrt{2}}, \frac{a}{\sqrt{2}} \right) \] よって、 \[ x_A = a + \frac{a}{\sqrt{2}} = \left( 1 + \frac{1}{\sqrt{2}} \right) a \] \[ y_A = \frac{21\pi}{4} a + \frac{a}{\sqrt{2}} = \left( \frac{21\pi}{4} + \frac{1}{\sqrt{2}} \right) a \] したがって、求める係数は次のとおりです。

答え：(キ) \(1 + \frac{1}{\sqrt{2}}\)（または \(\frac{2+\sqrt{2}}{2}\)）
答え：(ク) \(\frac{21\pi}{4} + \frac{1}{\sqrt{2}}\)（または \(\frac{21\pi + 2\sqrt{2}}{4}\) など）
Point

解答のポイント

  • 2体問題は重心と相対運動に分解する：外力が働かない系では重心は等速直線運動をし、相対運動は等速円運動になります。
  • 回転角の計算：初期位相と回転数に注意して、時刻 \(t\) を求めます。図からAとBのどちらがどの位置にあるかを正確に読み取ることが重要です。
  • 座標の合成：\(\vec{r_A} = \vec{r_G} + \vec{r_{A/G}}\) のベクトル和で座標を求めます。
